G&G Design System

Living style guide for the «Грим та Грім» site. Foundations are defined as CSS custom properties in css/variables.css; every component below is rendered with the real production classes. Written reference: DESIGN_SYSTEM.md.

Typography

One family — --font-family-base (Arial, Helvetica, sans-serif). Body text is --font-size-base / --line-height-base; display headings tighten to --line-height-tight (1.15).

Elevation

Shadows imply layer height: dropdowns sit above content, modals above everything. The accent glow is reserved for the primary CTA.

--shadow-dropdown menus, popovers
--shadow-modal modal dialogs
--shadow-glow-accent primary CTA
--shadow-glow-accent-hover primary CTA, hover

Motion

Standard transitions run --transition-speed (0.3s) with --transition-func (ease-in-out); softer reveals (gallery modal, expanding text) use --duration-slow (0.35s) with --ease-smooth. All animations collapse to 0.01s under prefers-reduced-motion.
